# dotfiles

## How To Use

- To install dotfiles on a new machine:

```shell
git clone git@github.com:m02uku/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install
```

- To update an existing installation:

```shell
cd ~/.dotfiles
git pull
./install
```
## Requirements

- Each lsp
- rg command (ripgrep)
- starship
